# 页面设计取色板

本文件作为当前页面设计的唯一取色依据。

后续页面设计与实现只能从以下颜色中取值，不允许新增其他主色、辅助色或强调色。

## 允许使用的颜色

| 用途 | 色值 | 说明 |
| --- | --- | --- |
| 品牌主高亮色 | `#F5A623` | 品牌最核心统一色。所有标题高亮、section 标签高亮统一使用该色。 |
| 浅灰区块色 | `#F2F4F3` | 用于左侧面板浅色背景替代原深色渐变。 |
| 纯白色 | `#FFFFFF` | 可直接作为左侧面板背景或常规白色底。 |
| 米白色 | `#F9F7F2` | 可保留用于捐赠表单右侧区域背景。 |

## 强制修改规则

1. 左侧面板原深色渐变背景必须替换为 `#F2F4F3` 或 `#FFFFFF`。
2. 所有标题高亮色中原有的 teal/green 必须统一替换为 `#F5A623`。
3. 捐赠表单右侧当前米白底 `#F9F7F2` 可以保留。
4. 捐赠表单右侧 section 标签，如 `YOUR DETAILS`、`PAYMENT DETAILS`，原青绿色必须统一替换为 `#F5A623`。

## 设计约束

- 页面设计只能使用本文件列出的 4 个颜色。
- 不允许从参考图之外继续扩展新颜色。
- 如需新增颜色，必须先更新本文件后才能使用。
